# Onderzoek — ORM AI Nieuws (DBR-rapport)

Werkmap voor het ontwerpgericht-onderzoek-rapport over ORM AI Nieuws. Doelpubliek: lectoraat Hanzehogeschool Groningen (Harald Pol) + externe publicatie.

## Mappen

- **`markdown/`** — bron-bestanden (één `.md` per hoofdstuk + samengevoegd rapport)
- **`word/`** — gegenereerde `.docx`-bestanden (open je in Word om te lezen)
- **`rapport-skelet.md`** → verplaatst naar `markdown/rapport-skelet.md` — overzichts-skelet, leidraad voor wat per sectie moet komen

## Voor Lex — hoe lees je dit?

Open de `.docx`-bestanden in `word/`. Per hoofdstuk één bestand. Na alle hoofdstukken klaar zijn, maak ik één samengevoegd `rapport-v0.1.docx` dat je in z'n geheel aan Harald kunt geven.
